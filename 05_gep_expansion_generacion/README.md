# Expansión de generación

Este bloque estudia decisiones de inversión en generación bajo demanda futura, reserva, ENS y restricciones ambientales.

## Estructura sugerida

- `modelos/`: formulaciones matemáticas.
- `datos/`: casos o datos propios del bloque.
- `notebooks/`: exploración y visualización.
- `actividades/`: consignas y rúbricas.
