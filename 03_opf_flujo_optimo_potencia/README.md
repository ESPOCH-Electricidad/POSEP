# Flujo óptimo de potencia

Este bloque relaciona optimización económica con restricciones físicas de red, usando OPF-DC y OPF-AC.

## Estructura sugerida

- `modelos/`: formulaciones matemáticas.
- `datos/`: casos o datos propios del bloque.
- `notebooks/`: exploración y visualización.
- `actividades/`: consignas y rúbricas.
