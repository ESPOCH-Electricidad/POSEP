# Módulo 04 — Flujo óptimo de potencia

El OPF introduce explícitamente la red eléctrica. La decisión de generación ya no depende solo de costos y demanda total, sino de ubicación, balances nodales, límites térmicos, ángulos de tensión, pérdidas y restricciones eléctricas.

## Resultado de aprendizaje

- Formular OPF DC como LP.
- Interpretar balance nodal, ángulo de referencia y límites de línea.
- Reconocer congestión y precios sombra.
- Comprender la diferencia conceptual entre OPF DC y OPF AC.
