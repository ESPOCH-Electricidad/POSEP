# Módulo 05 — Proyección de demanda y escenarios

Este módulo prepara uno de los insumos más importantes de operación y planificación: demanda de energía, demanda pico, perfiles temporales y escenarios.

## Resultado de aprendizaje

- Distinguir energía anual, demanda pico y perfil horario.
- Construir indicadores de factor de carga y crecimiento.
- Aplicar regresión y series de tiempo básicas.
- Construir escenarios bajo/base/alto.
- Exportar demanda a formatos utilizables por AMPL.
