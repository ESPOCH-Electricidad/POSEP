# Módulo 06 — Planificación de expansión de transmisión

Este módulo estudia decisiones de construcción o refuerzo de líneas. La red se representa mediante corredores existentes y candidatos, con costos de inversión, límites térmicos y balance nodal.

## Resultado de aprendizaje

- Formular modelos de transporte para expansión.
- Formular DC-TNEP con ángulos y flujos.
- Usar variables binarias para decisiones construir/no construir.
- Aplicar restricciones big-M de activación.
- Formular expansión multietapa.
