# Módulo 07 — Planificación de expansión de generación

Este módulo estudia decisiones de inversión en nueva capacidad de generación bajo demanda futura, reserva, bloques de carga y criterios económicos de largo plazo.

## Resultado de aprendizaje

- Formular GEP estático y multianual.
- Usar CRF para anualizar inversiones.
- Distinguir CAPEX, OPEX fijo, OPEX variable y costo de combustible.
- Usar LCOE como métrica auxiliar, no como reemplazo del modelo de sistema.
- Interpretar screening curves y factor de capacidad.
