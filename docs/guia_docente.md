# Guía docente

[Menú principal](../README.md)

Esta guía resume una forma de uso del material durante la clase y la práctica.

## Secuencia sugerida

1. Presentar el problema técnico del módulo.
2. Revisar la teoría y las ecuaciones del README.
3. Abrir uno de los ejemplos o modelos asociados.
4. Identificar datos, unidades y estructura de índices.
5. Pedir al estudiante que construya sus archivos de trabajo.
6. Resolver el caso y revisar factibilidad, balances, límites y costos.
7. Cerrar con interpretación técnica y sensibilidad.

## Archivos que debe producir el estudiante

En los módulos de optimización eléctrica se recomienda solicitar tres archivos: el modelo algebraico, el archivo de datos y el archivo de ejecución. En el módulo de demanda, el producto principal puede ser un cuaderno o script de Python, junto con tablas de salida para los módulos de expansión.

## Criterio de revisión

La revisión debe priorizar la correspondencia entre formulación y problema físico: unidades, balance de potencia o energía, límites operativos, dominio de variables, coherencia de costos y reproducibilidad del resultado.
