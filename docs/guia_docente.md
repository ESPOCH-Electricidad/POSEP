# Guía docente

[Menú principal](../README.md)

Esta guía resume el uso recomendado del repositorio en clase.

## Uso sugerido

1. Iniciar cada unidad con el README del módulo.
2. Discutir la teoría y las ecuaciones.
3. Revisar el ejemplo o modelo principal.
4. Construir el archivo de datos a partir de las tablas del caso.
5. Implementar el modelo en AMPL cuando la actividad lo requiera.
6. Resolver la actividad.
7. Solicitar un breve informe técnico.

## Enfoque

El repositorio separa:

- teoría;
- datos;
- ejemplos/modelos;
- actividades;
- casos integradores.

La implementación computacional se trabaja con archivos `.mod`, `.dat` y `.run` elaborados por el estudiante. La guía de apoyo está disponible en [Guía AMPL](guia_ampl.md).

No se publican soluciones completas ni archivos de salida generados por los estudiantes.
