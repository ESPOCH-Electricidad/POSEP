# Auditoría del paquete v16

## Cambios principales

- Reubicación del horizonte temporal en la presentación principal del curso.
- Redefinición del Módulo 01 como módulo de optimización matemática aplicada.
- Reubicación de contenidos económicos en Módulo 03, Módulo 06 y Módulo 07.
- Ampliación del Módulo 02 con control de flujo AMPL: `for`, `repeat while`, `if`, conexión Excel/CSV y exportación.
- Inclusión de modelos AMPL ejecutables como plantillas docentes.
- Inclusión de scripts Python para generación de datos, exportación y análisis de resultados.
