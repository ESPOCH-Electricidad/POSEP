# Evaluación propuesta

## Rúbrica para interpretación de un caso de planificación

| Criterio | Excelente | Aceptable | Insuficiente |
|---|---|---|---|
| Identificación de datos | Reconoce demanda, tecnologías, límites, costos y restricciones | Reconoce la mayoría de datos relevantes | Confunde datos o ignora parámetros clave |
| Formulación conceptual | Relaciona variables, función objetivo y restricciones | Describe parcialmente el modelo | No conecta datos con el modelo |
| Interpretación de resultados | Explica inversión, despacho, ENS, reserva y costo | Interpreta algunos resultados | Solo transcribe números |
| Análisis de sensibilidad | Compara escenarios y justifica cambios | Compara sin suficiente justificación | No identifica efectos de cambios |
| Uso crítico de IA | Detecta errores y los corrige con fundamento técnico | Detecta errores, pero argumenta poco | Acepta la respuesta de IA sin revisión |
