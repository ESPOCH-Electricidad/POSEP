# Estructura pedagógica v16

## Decisión central

La asignatura se organiza por capas:

1. **Mapa general del curso**: ubicación de los problemas en el horizonte temporal.
2. **Base matemática**: optimización antes de cualquier aplicación eléctrica.
3. **Base computacional**: AMPL como lenguaje algebraico y Python/Excel como soporte de datos.
4. **Operación**: despacho, unit commitment, OPF y señales marginales.
5. **Planificación**: demanda, expansión de transmisión y expansión de generación.

## Criterio para redactar teoría útil

Cada sección debe responder cuatro preguntas:

1. ¿Qué decisión técnica se está modelando?
2. ¿Qué variables de decisión representan esa decisión?
3. ¿Qué restricciones físicas, operativas o económicas limitan la solución?
4. ¿Cómo se valida si el resultado tiene sentido?

No se incorporan figuras puramente decorativas. Las figuras deben explicar una relación técnica: región factible, orden de mérito, curva de duración, balance nodal, flujo DC, big-M, expansión multietapa o screening curve.
