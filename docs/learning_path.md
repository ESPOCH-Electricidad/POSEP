# Learning Path

## Phase 1 — Mathematical foundations
Linear programming, indexing, and modeling structure.

## Phase 2 — System operation
Economic dispatch, hydrothermal coordination, and unit commitment.

## Phase 3 — Network optimization
DC and AC optimal power flow.

## Phase 4 — Expansion planning
Transmission (TNEP) and generation (GEP).

## Integration principle
All models converge in case studies such as Garver and IEEE test systems.
