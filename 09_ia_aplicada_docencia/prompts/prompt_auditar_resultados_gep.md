# Prompt: generar explicación para auditar

Genera una explicación breve de los resultados de un modelo GEP para el caso Garver. Incluye intencionalmente tres errores técnicos sutiles relacionados con:

1. Confusión entre capacidad construida y despacho.
2. Interpretación incorrecta de ENS.
3. Capacidad firme de renovables.

Luego, en una sección separada, lista los errores esperados para que el docente pueda preparar la guía de corrección.
