# Planificación y Operación de Sistemas Eléctricos de Potencia

![Portada académica](assets/img/portada_academica_v15.png)

Este repositorio reúne el material de trabajo de la asignatura. La ruta inicia con la formulación matemática de problemas de optimización, continúa con su implementación en AMPL y avanza hacia aplicaciones de operación y planificación de sistemas eléctricos: despacho económico, flujo óptimo de potencia, proyección de demanda, expansión de transmisión y expansión de generación.
