# Planificación y Operación de Sistemas Eléctricos de Potencia

![Banner del repositorio](docs/assets/img/banner_operacion_planificacion_sep.svg)

Repositorio académico de apoyo a la asignatura **Planificación y Operación de Sistemas Eléctricos de Potencia**. El material está organizado como un curso navegable: teoría por tema, modelos matemáticos, casos de estudio, notebooks de apoyo y actividades aplicadas.

[Menú principal](README.md) · [Ruta de aprendizaje](docs/ruta_aprendizaje.md) · [Mapa de modelos](docs/mapa_modelos.md) · [Mapa de casos](docs/mapa_casos.md) · [Evaluación](docs/evaluacion.md)

## Bienvenida al curso

El repositorio sigue la ruta:

```text
Fundamentos de optimización → Operación → OPF → TNEP → GEP → Casos de estudio
```

La intención es que el estudiante no entre directamente a archivos sueltos. Cada bloque inicia con una presentación conceptual del tema, después guía hacia modelos, casos y actividades.

## Cómo usar el repositorio

1. Iniciar en el README del bloque.
2. Revisar la teoría y las ecuaciones principales.
3. Abrir el mapa de modelos del bloque.
4. Seleccionar el caso recomendado.
5. Resolver la actividad correspondiente.
6. Volver al bloque para interpretar resultados.
