# Instalación y uso de AMPL

## Uso local

El docente indicará la versión de AMPL y solver a utilizar. Los archivos `.mod` y `.run` pueden distribuirse mediante la plataforma institucional.

## Uso con Google Colab

Una alternativa para laboratorios virtuales es usar `amplpy` en Google Colab. En ese caso, el notebook puede instalar AMPL, cargar datos y resolver modelos bajo las condiciones de licencia correspondientes.

## Recomendación docente

Para el repositorio público, mantener:

- datos `.dat`;
- datos `.csv`;
- notebooks de exploración;
- guías de interpretación.

Reservar para la plataforma institucional:

- modelos `.mod`;
- ejecución `.run`;
- soluciones completas;
- claves de evaluación.
